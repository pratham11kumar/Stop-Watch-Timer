# ⏱ Stopwatch & Timer App

A React-based Stopwatch and Timer application built for Web Dev Cohort 2026. The app provides stopwatch and countdown timer functionality with a clean UI, responsive design, and smooth user experience.

---

## 🚀 Features

### Stopwatch
- Start stopwatch
- Pause stopwatch
- Reset stopwatch
- HH:MM:SS time format

### Timer
- Custom time input
- Start countdown
- Pause countdown
- Reset timer
- Clear time display

### UI & UX
- Responsive design
- Bhagwa + Light Green theme
- Clean component structure
- Mobile friendly interface

---

## 🛠 Tech Stack

- React
- JavaScript
- CSS
- Vite

## ⚙ Installation

Clone repository:

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

Go into project folder:

```bash
cd StopWatchTimer
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
